DOLOS

Dolos is a CleverBot-esque artificial intelligence contained in under 200 lines of code. It draws from a conversation data file to generate responses to user input, and actively learns from how the user talks to it. Theoretically, the more data Dolos has, the more accurately it will respond to input, and the more human it will seem.
