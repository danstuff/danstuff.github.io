Locker is a small piece of software that can be used to encrypt a text document. It uses XOR encryption to obscure data in a way that is entirely secure and reversible. To use Locker, simply type whatever you want in "locker.txt," then start up the executable and enter a key. The file will now be unreadable, but restart the executable and re-enter your key, and the file will be decrypted, allowing you to access your data. Created in the Spring of 2016.
