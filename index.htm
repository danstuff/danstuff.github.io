Space Crusade

Space Crusade is a small arcade-style shooter in which you defend against an onslaught of alien spaceships using your lasers and missiles. It features customizable ship colors, three different types of enemy ships, power-ups, and more! Aim your ship with the mouse and move using WASD. Left click to shoot lasers, and right click to shoot missiles. Navigate menus with the mouse. Created in the Summer of 2014.

Cacophone

Cacophone is a game about making music using swinging pendulums. There are multiple types of pendulum, accessible with the mouse wheel, and stationary blocks which add an additional layer of fun. It also features a flexible settings menu. Created in the summer of 2016.

Download | Github Page

Obama Trail

Obama Trail is a short quiz game themed like the classic educational game Oregon Trail. By answering the questions as accurately as you can to what Obama actually did, you'll help Obama and Biden survive their presidential term! Created in April of 2017 with the help of writers Kami Beckford, Myia Samuels, and Mateo Ginocchi.

Download | Github Page

CorpsLog

CorpsLog is a flexible, cross-platform timesheet app designed to interface with the OnCorps Reports timesheet tracking service. It improves the accuracy of worker's submitted data with the help of location tracking, notification reminders to punch in and punch out, and automatic submission of hours to the OnCorps website. This is an ongoing project that was started in January, 2018.
