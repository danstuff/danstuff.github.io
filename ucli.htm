USLESS CODING LANGUAGE INTERPRETER

Useless Coding Language Interpreter (UCLI) is a bytecode interpreter which can compile and run programs written in a stack-based programming language called UCL. It was built entirely from the ground up and features input, output, and basic arithmetic. As the name suggests, the language is fairly rudimentary. Created late in 2014.
