INSTALINK

InstaLink is a simple program designed purely for personal convenience; its aim is to make it easier to install essential programs on a new computer. The user can add and remove website URLs from its list, and open all of those URLs at once in the default internet browser. It comes preloaded with some of my favorite programs' websites. Created in the Summer of 2016.
